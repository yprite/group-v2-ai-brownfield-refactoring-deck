# group-v2 AI Brownfield Refactoring Deck

Browser presentation deck for `group-v2` AI-assisted brownfield refactoring.

## Files

- `index.html`: standalone 26-slide HTML deck.
- `screenshots/slide-01.png` through `screenshots/slide-26.png`: rendered slide screenshots.
- `screenshots/contact-sheet-01-12.png`: overview of slides 1-12.
- `screenshots/contact-sheet-13-24.png`: overview of slides 13-24.
- `screenshots/contact-sheet-25-26.png`: overview of slides 25-26.
- `videos/slide-01-marvel-title-intro.mp4`: captured slide 1 title-mask intro.

## Controls

- Next: Right arrow, Space, PageDown
- Previous: Left arrow, PageUp
- Speaker notes: N
- Fullscreen: F
- Print/PDF: P

## Validation

Rendered and checked all 26 slides in Chrome at:

- 1600x900
- 1280x720
- 1024x768
- 1440x900

The deck scales as a single 1600x900 artboard to avoid projector and browser viewport clipping.
